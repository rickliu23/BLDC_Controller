# 2 保护情况说明
## 2.1 形式说明
+ 无论是哪种保护，都是在检测到电流异常后，做出相对应的保护措施

## 2.2 保护类型目录
+ 电机堵转
+ 相间短路
+ 功率管击穿
+ 反电动势异常回馈
+ 持续过载运行
+ 启动过流
+ 缺相运行


# 3 高侧、低侧采样的区别与优缺点

## 3.1 功能差别
|   排名  | 功能            | 核心场景        |       高侧能否实现       |                低侧能否实现                |
| :---: | :------------ | :---------- | :----------------: | :----------------------------------: |
| **1** | **检测对地短路**    | 安全保护、故障诊断   |       ✅ **能**      |          ❌ **不能**（电流绕开采样电阻）          |
| **2** | **检测对电源短路**   | 安全保护、故障诊断   |      ❌ **不能**      |                ✅ **能**               |
| **3** | **PWM同步采样窗口** | 日常控制稳定性     |     ⚠️ 低占空比时受限     |              ⚠️ 高占空比时受限              |
| **4** | **抗接地干扰/地环路** | 汽车、工业、分布式系统 | ✅ **能**（参考母线，不依赖地） |          ❌ **不能**（受地电位波动影响）          |
| **5** | **负载参考地抬高**   | 多板系统、通信接口   |    ✅ **能**（无影响）    | ❌ **不能**（$I \times R_{shunt}$ 抬高地电位） |
| **6** | **低侧制动电流检测**  | 制动能量回收、保护   |      ❌ **不能**      |                ✅ **能**               |
| **7** | **高侧制动电流检测**  | 高端制动策略      |   ✅ **能**（独立桥臂采样）  |               ❌ **不能**               |
| **8** | **共模电压要求**    | 器件选型（成本相关）  |   ⚠️ 要求高（接近母线电压）   |           ✅ **低**（接近地，易实现）           |
